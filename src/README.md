# Vercel

Manage Vercel projects, deployments, domains, environment variables, logs, teams, checks, aliases, and secrets via the Vercel REST API.

## Categories

- Cloud Infrastructure
- Developer Tools

## Credentials

| Key | Label | Kind |
|-----|-------|------|
| `token` | API Token | password |
| `team_id` | Team ID (optional) | text |

## Installation

This skill is included in [EnterpriseAgentOS](https://officeos.co) and requires no separate installation.
